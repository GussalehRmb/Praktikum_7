# DISKRIPSI
Membuat program sederhana dengan mengaplikasikan penggunaan class dengan menggunakan ketentuan fitur program tambah data, menghapus data berdasarkan nama, dan mengubah data berdasarkan nama.
## Fitur-fitur pada bagian program
<pre>1. Tambah()
Digunakan untuk menambahkan data mahasiswa baru.
User akan memasukkan:
<li>Nama mahasiswa
<li>Nilai mahasiswa
data kemudian tersimpan</li></pre>
<pre>2. Tampilkan()
Menampilkan semua data mahasiswa yang sudah tersimpan.
Jika belum ada data, program menampilkan pesan â€œBelum ada dataâ€.</pre>  
<pre>3. hapus(nama)
Menghapus data mahasiswa berdasarkan nama yang dimasukkan oleh user.
Jika nama tidak ditemukan, program menampilkan pesan error</pre>
<pre>4. ubah(nama)
Mengubah nilai mahasiswa berdasarkan nama.
Jika nama ditemukan, nilai baru diminta dan data diperbarui.</pre>
